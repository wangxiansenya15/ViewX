# ViewX v0.12.20-dev 发布说明

## 📅 发布日期
2025-12-20

## � 版本摘要
**v0.12.20-dev** 是 ViewX 平台的一个重要里程碑版本，标志着**基础功能开发已基本完成**。此版本构建了完整的后端微服务架构和前端交互界面，集成了先进的 AI 能力、实时通讯系统以及高标准的安全防护机制。

---

## ✨ 核心功能完成度概览

### 1. � 用户与认证系统 (100% 完成)
- **多方式登录**: 支持邮箱/密码登录，以及 GitHub OAuth2 第三方登录。
- **JWT 鉴权**: 实现了基于 JWT 的无状态认证，配合 Redis 黑名单机制处理注销。
- **人机验证**: 集成了 Turnstile、reCAPTCHA 和自定义滑块验证，保护登录注册接口。
- **权限控制**: 基于 RBAC 的角色权限管理 (User/Admin)，包含管理员后台。
- **个人中心**: 支持用户资料修改、头像上传（本地存储/对象存储兼容设计）。

### 2. 📹 视频内容系统 (90% 完成)
- **视频上传**: 支持大文件分片上传流程（前端组件 + 后端接口）。
- **视频处理**: 集成 FFmpeg 进行视频转码、自动封面提取。
- **播放体验**: 实现了统一的视频播放器控制条，支持倍速、音量记忆、剧场模式。
- **内容管理**: 视频审核流程（管理员后台）、视频列表展示、瀑布流布局。

### 3. 💬 社交与互动系统 (95% 完成)
- **实时聊天**: 基于 WebSocket 的私信系统，支持消息持久化和历史记录。
- **互动反馈**: 点赞、收藏、关注功能，基于 Redis 实现高性能计数。
- **评论系统**: 支持嵌套评论（楼中楼），实时更新。
- **通知中心**: 实现了消息通知、系统通知的推送与状态同步。

### 4. 🧠 AI 与搜索智能化 (80% 完成)
- **AI 集成**: 接入 Spring AI 和 DashScope (Qwen-Max) 模型。
- **语义搜索**: 利用 PostgreSQL `pgvector` 实现基于向量的语义搜索能力。
- **内容理解**: 视频内容的 AI 摘要生成和自动打标。
- **推荐框架**: 完成了推荐系统的基础架构设计 (`RecommendController/Service`)。

### 5. 🛡️ 安全与运维 (100% 完成)
- **依赖升级**: 全面修复了 Tomcat、Spring Framework、Netty 等组件的高危 CVE 漏洞。
- **Servlet 规范**: 迁移至 Jakarta EE (Spring Boot 3 标准)。
- **代码规范**: 建立了 `.cursorrules` 团队开发规范和统一返回结构 `Result<T>`。
- **监控**: 集成 Swagger/OpenAPI 文档，具备基本的日志记录。
