# 国际化文档 (Internationalization)

本项目使用 `vue-i18n` (v9) 进行多语言管理。

## 配置位置
- 配置文件: `src/i18n/index.ts`
- 语言包目录: `src/locales/`
  - `zh-CN.json`: 简体中文
  - `en-US.json`: 英文

## 使用方法

### 1. 在组件中使用
使用 Composition API 的 `useI18n` 钩子：

```typescript
<script setup lang="ts">
import { useI18n } from 'vue-i18n'

const { t } = useI18n()
</script>

<template>
  <div>{{ t('settings.title') }}</div>
</template>
```

### 2. 添加新词条
在 `src/locales/zh-CN.json` 和 `src/locales/en-US.json` 中保持结构一致：

**zh-CN.json**:
```json
{
  "sidebar": {
    "home": "首页推荐"
  }
}
```

**en-US.json**:
```json
{
  "sidebar": {
    "home": "Home"
  }
}
```

## 当前覆盖范围
目前已完成以下模块的国际化：
- **设置页面 (Settings)**: 标题、选项、按钮文本。
- **侧边栏 (Sidebar)**: 导航菜单项、Gemini 推广卡片。
- **启动屏 (Splash Screen)**: Slogan 文本。
