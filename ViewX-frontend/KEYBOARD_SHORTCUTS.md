# ViewX 视频播放器快捷键

## 播放控制

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **空格键** | 播放/暂停 | 切换视频播放状态 |
| **M** | 静音/取消静音 | 快速切换音量 |
| **F** | 全屏/退出全屏 | 切换全屏模式 |
| **U** | 清屏/剧场模式 | 隐藏所有 UI 元素 |

## 进度控制

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **← 左箭头** | 快退 5 秒 | 连续按可连续后退 |
| **→ 右箭头** | 快进 5 秒 | 连续按可连续前进 |

## 视频导航

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **↑ 上箭头** | 上一个视频 | 切换到上一个视频 |
| **↓ 下箭头** | 下一个视频 | 切换到下一个视频 |

## 互动功能

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **Z** | 点赞 | 为当前视频点赞 |
| **双击视频** | 点赞 | 双击视频区域点赞 |

## 鼠标控制

| 操作 | 功能 | 说明 |
|------|------|------|
| **点击进度条** | 跳转到指定位置 | 精确控制播放进度 |
| **拖动进度点** | 拖动调整进度 | 实时预览 |
| **悬停音量图标** | 显示音量滑块 | 垂直滑块调节 |
| **悬停倍速按钮** | 显示速度选项 | 6 档速度可选 |
| **点击音量图标** | 快速静音 | 一键静音/取消 |
| **点击全屏按钮** | 切换全屏 | 进入/退出全屏模式 |
| **点击播放顺序** | 切换播放模式 | 顺序/循环/随机 |
| **点击清屏按钮** | 剧场模式 | 隐藏所有 UI |

## 播放模式

- **顺序播放** - 按顺序播放视频列表
- **循环播放** - 单个视频循环播放
- **随机播放** - 随机顺序播放

## 倍速选项

- **0.5x** - 慢速播放
- **0.75x** - 较慢播放
- **1.0x** - 正常速度（默认）
- **1.25x** - 稍快播放
- **1.5x** - 快速播放
- **2.0x** - 超快播放

## 注意事项

1. **快捷键仅在当前激活的视频上生效**
   - 推荐流：当前可见的视频
   - 网格模式：鼠标悬停的视频

2. **输入框中不响应快捷键**
   - 评论输入时不会触发快捷键
   - 搜索框中不会触发快捷键

3. **音量和倍速设置全局共享**
   - 所有视频使用相同的音量设置
   - 所有视频使用相同的播放速度
   - 设置会自动保存，刷新后保持

4. **控制条自动隐藏**
   - 播放时 3 秒无操作自动隐藏
   - 鼠标移动时自动显示
   - 暂停时始终显示
   - 悬停在控制条上时不隐藏

5. **全屏模式**
   - 支持 F 键快捷键
   - 点击全屏按钮
   - ESC 键退出全屏（浏览器默认）

6. **剧场模式/清屏**
   - U 键快速切换
   - 隐藏所有 UI 元素，专注观看
   - 再次按 U 键恢复

## 提示

💡 **快速静音**：按 M 键或点击音量图标

💡 **精确跳转**：点击进度条任意位置

💡 **连续快进**：按住右箭头键不放

💡 **快速调速**：鼠标悬停在"倍速"按钮上选择

💡 **全屏观看**：按 F 键或点击全屏按钮

💡 **视频切换**：使用上下箭头键快速切换视频

💡 **快速点赞**：按 Z 键或双击视频

💡 **沉浸观看**：按 U 键进入清屏模式
